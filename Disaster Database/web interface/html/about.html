About

Welcome to the about page. This is where you learn about the purpose of this website and the database.


What is this website?

This website is an interface that allows interaction with the disaster database.


What is the database?

This database is a collection of information about disaters worldwide, both natural and manmade.


Why disasters?

The goal when creating this database was to try and find patterns. The idea is that with enough information, we will be able to predict disasters more accurately and know how to survive them. Hopefully, this database will be able to save lives.
